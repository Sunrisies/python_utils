# Rust编程语言

由 Steve Klabnik、Carol Nichols 和 Chris Krycho 编写，得到了 Rust 社区的贡献支持。

此版本的文本假设您正在使用 Rust 1.90.0（发布日期：2025-09-18）或更高版本。在所有项目的 *Cargo.toml* 文件中设置 `edition = "2024"`，以配置它们使用 Rust 2024 版本的特性。有关安装或更新 Rust 的说明，请参阅 [“Installation” section
of Chapter 1][install]<!-- ignore -->。有关版本的更多信息，请参阅 [Appendix E][appendix-e]<!-- ignore -->。

HTML格式可以在以下网址在线获取：
[https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/)
此外，在安装了 `rustup`; 的Rust系统上，可以通过运行 `rustup doc
--book` 来离线访问该格式。

还有一些社区的[翻译版本]可供参考。

这段文字在 [paperback and ebook format from No Starch
Press][nsprust] 中可用。
