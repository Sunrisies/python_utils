# 函数式语言特性：迭代器和闭包

Rust的设计借鉴了许多现有语言和技术的经验，其中对Rust影响较大的就是_函数式编程_。函数式编程的特点通常是将函数作为值来使用，比如通过参数传递函数，从其他函数返回函数，或将函数赋值给变量以便后续执行等等。

在本章中，我们不会讨论什么是函数式编程，而是会探讨一些与许多语言中的函数式特性相似的Rust特性。

更具体地说，我们将涵盖以下内容：

- **闭包**：一种类似函数的构造，可以存储在变量中。
- **迭代器**：一种处理一系列元素的方法。
- 如何在第12章中使用闭包和迭代器来改进I/O操作。
- 闭包和迭代器的性能（剧透警告：它们的性能比你想象的要快！）

我们已经介绍了一些其他Rust特性，比如模式匹配和枚举，这些特性同样受到函数式编程风格的影响。因为掌握闭包和迭代器是编写高效、符合Rust惯用的代码的重要部分，所以我们会将这一整章的内容都集中在这两个主题上。
