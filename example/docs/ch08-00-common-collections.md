# 常见集合

Rust的标准库包含许多非常有用的数据结构，称为“collections”。大多数其他数据类型只能表示一种特定的值，而collections则可以包含多个值。与内置的数组和元组类型不同，这些collections所指向的数据存储在堆上，这意味着数据量在编译时不需要预先知道，并且可以在程序运行过程中增加或减少。每种collections都有不同的功能和成本，选择适合当前情况的collections是一种需要随着时间逐渐掌握的技能。在本章中，我们将讨论在Rust程序中经常使用的三种collections：

- 向量允许你将多个值并排存储在一起。  
- 字符串是由字符组成的集合。之前我们已经提到过 `String` 类型，但在这一章中，我们将更深入地讨论它。  
- 哈希映射允许你将一个值与特定的键关联起来。这是一种更通用的数据结构——映射的特化实现方式。

要了解标准库提供的其他类型的集合，请参阅[相关文档][collections]。

我们将讨论如何创建和更新向量、字符串以及哈希映射，同时也会探讨这些因素各自的特点。
