# 高级功能

到目前为止，你已经了解了Rust编程语言中最常用的部分。在继续进行下一个项目之前，在第二章中，我们将探讨一些你可能会偶尔遇到但可能不会每天使用的语言特性。你可以将这一章作为遇到任何未知情况的参考。这里介绍的功能在非常特定的情况下非常有用。虽然你可能不会经常使用这些功能，但我们希望确保你完全掌握了Rust所提供的所有功能。

在本章中，我们将涵盖以下内容：

- 不安全的 Rust：如何放弃 Rust 的一些保证，并负责手动维护这些保证
- 高级特性：相关类型、默认类型参数、完全限定的语法、超特性以及与特性相关的新类型模式
- 高级类型：关于新类型模式的更多信息、类型别名、永不类型，以及动态大小的类型
- 高级函数和闭包：函数指针和返回闭包
- 宏：在编译时定义更多代码的几种方式

这是一套涵盖多种 Rust 功能的工具集，适合各种需求！让我们开始吧！
