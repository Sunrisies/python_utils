# 关于 Cargo 和 Crates.io 的更多信息

到目前为止，我们仅使用了 Cargo 中最基础的功能来构建、运行和测试我们的代码，但实际上 Cargo 的功能远不止这些。在本章中，我们将介绍一些更高级的功能，以展示如何完成以下操作：

- 通过发布配置文件自定义构建过程。  
- 在 [crates.io](https://crates.io/) 上发布库文件。  
- 使用工作区来组织大型项目。  
- 从 [crates.io](https://crates.io/) 安装二进制文件。  
- 通过自定义命令扩展 Cargo 的功能。

Cargo 的功能远不止我们在本章中介绍的内容，因此如果你想全面了解它的所有功能，请参考其文档：[点击这里](https://doc.rust-lang.org/cargo/)获取更多信息。
