# 面向对象编程的特点

<!-- Old headings. Do not remove or links may break. -->

<a id="object-oriented-programming-features-of-rust"></a>

面向对象编程（OOP）是一种编程模型。作为编程概念的对象最早出现在20世纪60年代的Simula语言中。这些对象对Alan Kay的编程架构产生了影响，该架构中对象之间会互相传递消息。为了描述这一架构，他在1967年提出了“面向对象编程”这一术语。许多不同的定义都描述了什么是OOP，根据某些定义，Rust是面向对象的语言，而根据其他定义则不是。在本章中，我们将探讨一些通常被认为是面向对象的特征，以及这些特征如何体现在Rust的惯用法中。接下来，我们将向您展示如何在Rust中实现面向对象的设计模式，并讨论这样做与利用Rust的一些优势来解决问题之间的权衡。
