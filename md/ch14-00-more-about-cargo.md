# 更多关于Cargo和Crates.io的信息

到目前为止，我们仅使用了Cargo中最基本的功能来构建、运行和测试我们的代码，但实际上它还能做更多的事情。在本章中，我们将讨论一些更高级的功能，以展示如何执行以下操作：

- 通过发布配置文件来定制你的构建过程。  
- 在[crates.io](https://crates.io/)上发布库文件。  
- 使用工作区来组织大型项目。  
- 从[crates.io](https://crates.io/)安装二进制文件。  
- 通过自定义命令来扩展Cargo的功能。

Cargo的功能远不止我们在本章中介绍的那些功能。如需了解其所有特性的完整说明，请参阅[其文档](https://doc.rust-lang.org/cargo/)。
