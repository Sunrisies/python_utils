# 高级功能

到目前为止，你已经学习了Rust编程语言中最常用的部分。在继续下一个项目之前，在第21章中，我们将探讨一些你可能会偶尔遇到但可能不会每天使用的语言特性。你可以将这一章节作为遇到未知问题的参考。这里介绍的功能在非常特定的情况下非常有用。虽然你可能不会经常使用它们，但我们希望确保你能够掌握这些知识。掌握Rust所提供的所有功能。

在本章中，我们将涵盖以下内容：

- 不安全的Rust：如何放弃Rust的一些保证，并负责手动维护这些保证  
- 高级特性：相关类型、默认类型参数、完全限定的语法、超特性以及与特性相关的新类型模式  
- 高级类型：关于新类型模式的更多信息、类型别名、永不类型以及动态大小的类型  
- 高级函数和闭包：函数指针和返回闭包- 宏：在编译时定义能够进一步定义更多代码的代码的方式

这是一套丰富的Rust功能，适合各种需求！让我们开始吧！
