Rust 文档

欢迎来到由 Rust 项目提供的文档概述页面。此页面包含各种有用参考链接，其中大多数都可以离线使用（如果使用 rustup doc 打开）。这些资源中有许多是以“书籍”的形式呈现的；我们将其统称为“Rust 书架”。有些书籍篇幅较长，有些则较短。

所有这些书籍都由 Rust 组织管理，但这里也包含了其他非官方的文档资源！

如果您只是想查找标准库参考，那么这里就有： Rust API 文档

§学习 Rust

如果您想学习 Rust，那么这个部分非常适合您！所有这些资源都假设您之前已经编程过，但不是使用任何特定的语言：

§Rust 编程语言

这本被亲切地称为“书籍”的书 《Rust 编程语言》将从基础开始为您介绍这门语言。您将在这个过程中构建一些项目，最终能够熟练掌握如何使用这门语言。

§示例驱动的学习 Rust

如果您不喜欢阅读数百页关于一门语言的资料，那么 《示例驱动的学习 Rust》适合您。RBE 展示了大量代码，同时提供了练习！

§Rustlings

Rustlings指导您下载并设置 Rust 工具链，然后提供一个交互式工具，帮助您解决 Rust 编程挑战。

§Rust 游乐场

Rust 游乐场是一个很好的地方，可以尝试和分享一些小代码片段，或者实验一些最受欢迎的库。

§使用 Rust

一旦您熟悉了这门语言，这些资源可以帮助您将其应用于实际工作中。

§标准库

Rust 的标准库有 详细的 API 文档，其中解释了如何正确使用各种功能，以及完成各种任务的示例代码。代码示例在悬停时会有“运行”按钮，可以打开游乐场中的示例。

§您的个人文档

每当您在一个 crate 中工作时，cargo doc --open 会生成您的项目的文档以及所有依赖项的文档，并以正确的版本显示在浏览器中。添加标志 --document-private-items 可以显示未标记为 pub 的项目。

§Rust 版本历史

发布说明描述了 Rust 工具链和语言的变更历史。

版本指南描述了 Rust 的版本及其差异。最新版本的工具链支持所有历史版本。

§rustc 手册

rustc 手册描述了 Rust 编译器 rustc。

§Cargo 手册

Cargo 手册是 Cargo 的指南，Cargo 是 Rust 的构建工具和依赖管理器。

§Rustdoc 手册

Rustdoc 手册描述了我们的文档工具 rustdoc。

§Clipy 手册

Clipy 手册描述了我们的静态分析器 Clipy。

§扩展错误列表

Rust 的许多错误都带有错误代码，您可以通过 rustc --explain 请求编译器进行扩展诊断。如果您愿意，也可以在这里阅读这些代码：rustc 错误代码

§掌握 Rust

一旦您对这门语言非常熟悉，这些高级资源可能会很有用。

§参考手册

参考手册不是正式的规范，但它比书籍更详细、更全面。

§风格指南

Rust 风格指南描述了 Rust 代码的标准格式。大多数开发者使用 cargo fmt 来调用 rustfmt 并自动格式化代码（结果符合此风格指南）。

§Rustonomicon

Rustonomicon是您了解不安全 Rust 的指南。它有时也被称为“nomicon”。

§不稳定书籍

不稳定书籍包含了不稳定功能的文档。

§rustc 开发指南

rustc-dev-guide 描述了编译器的工作原理以及如何贡献代码。如果您想从源代码构建或修改 Rust 编译器（例如，针对非标准目标），这将非常有用。

§专用 Rust

当在特定领域使用 Rust 时，可以考虑使用以下针对每个领域的资源。

§嵌入式系统

在开发裸机或嵌入式 Linux 系统时，您可能会发现由 嵌入式工作组维护的资源很有用。

§嵌入式 Rust 书籍

嵌入式 Rust 书籍面向那些熟悉嵌入式开发和 Rust，但尚未使用 Rust 进行嵌入式开发的开发者。
