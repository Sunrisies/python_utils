Rust编程语言帮助你编写更快、更可靠的软件。  
在编程语言设计中，高级的用户友好性与低级的控制功能往往存在矛盾；Rust则挑战了这种矛盾。通过平衡强大的技术能力和出色的开发者体验，Rust让你能够在不经历传统控制方式所带来的麻烦的情况下，直接控制低级细节（如内存使用）。

## Rust 适用于哪些场景

Rust 因多种原因而受到许多人的青睐。让我们来看看其中最重要的几个群体。

### 开发团队

Rust 被证明是大型开发团队之间协作的有效工具，无论团队成员在系统编程方面的经验如何。低级别代码容易存在各种细微的错误，而在大多数其他语言中，这些错误只能通过广泛的测试和经验丰富的开发人员的仔细代码审查来发现。在 Rust 中，编译器扮演着把关者的角色，它拒绝编译包含这些难以发现的错误的代码，包括并发相关的错误。通过与编译器协同工作，团队可以将精力集中在程序的逻辑上，而不是去修复那些错误。

Rust还为系统编程领域带来了现代的开发工具：

- Cargo是一个内置的依赖管理器和构建工具，能够轻松且一致地为整个Rust生态系统添加、编译和管理依赖项。
- `rustfmt`格式化工具确保了所有开发者使用一致的编码风格。
- Rust语言服务器为集成开发环境（IDE）提供了代码补全功能和内嵌式错误信息。

通过使用Rust生态系统中的这些工具以及其他工具，开发者在编写系统级代码时能够更加高效。

### 学生

Rust 适合那些对系统概念感兴趣的学生以及想要学习相关知识的人员。通过使用 Rust，许多人能够了解诸如操作系统开发等主题。该社区非常热情友好，乐于解答学生们的问题。通过像这本书这样的努力，Rust 团队希望让更多人对系统概念有所了解，尤其是那些刚开始学习编程的人。

### 公司

数百家大小公司都在生产环境中使用Rust来完成各种任务，包括命令行工具、网络服务、DevOps工具、嵌入式设备、音频和视频分析与转码、加密货币、生物信息学、搜索引擎、物联网应用、机器学习，甚至Firefox网络浏览器的许多核心部分。

### 开源开发者

Rust 是专为那些想要构建 Rust 编程语言、社区以及开发工具和相关库的人而设计的。我们非常希望你能为 Rust 语言的开发做出贡献。

### 重视速度和稳定性的人

Rust 是为那些渴望语言在速度和稳定性方面具有优势的人而设计的。这里的“速度”既包括 Rust 代码运行的速度，也包括 Rust 允许编写程序的速度。Rust 编译器的检查功能通过添加功能和重构来确保代码的稳定性。这与那些没有这些检查功能的脆弱旧语言中的代码形成对比，后者往往让开发者难以进行修改。通过追求零成本抽象——即那些可以像手动编写的代码一样快速编译成较低级别代码的高级功能——Rust 致力于使安全代码也能变得高效。

Rust语言希望也能支持更多的用户；这里提到的只是其中一些最重要的利益相关者。总体而言，Rust最大的目标是通过提供安全性与生产力、速度与人体工程学，消除程序员多年来不得不接受的权衡。试试Rust吧，看看它的选择是否适合你。

## 本书的目标读者

这本书假设您已经使用其他编程语言编写过代码，但并没有对具体的编程语言做出任何限制。我们努力使这些内容能够广泛地被来自不同编程背景的人所理解。我们没有花太多时间讨论什么是编程，或者应该如何思考编程问题。如果您是完全新的编程爱好者，那么阅读一本专门介绍编程入门知识的书籍会更适合您。

## 如何使用本书

一般来说，本书假设你按照从前到后的顺序阅读。后面的章节会基于前面章节中的概念进行展开，而前面的章节可能不会详细讨论某个特定主题，但会在后面的章节中再次提及该主题。

这本书包含两种类型的章节：概念性章节和项目性章节。在概念性章节中，你将学习Rust的某个方面。在项目性章节中，我们将一起构建小型程序，运用迄今为止所学的知识。第2章、第12章和第21章属于项目性章节；其余章节则属于概念性章节。

**第1章**介绍了如何安装Rust语言，如何编写“Hello, world!”程序，以及如何使用Cargo——Rust的包管理和构建工具。**第2章**则是对用Rust编写程序的实践性介绍，你将通过实现一个数字猜谜游戏来学习相关技巧。在这里，我们会以较高的层次讲解各种概念，后续章节会提供更多的细节说明。如果你想要立即动手实践，那么第2章就是适合你的地方。如果你是一个特别注重细节的学习者，希望在继续学习之前先了解每一个细节，那么你可以跳过第2章，直接进入**第3章**。第3章将介绍与其它编程语言类似的Rust特性；当你想要运用所学知识来开发一个项目时，再回到第2章进行学习即可。

在**第4章**中，你将了解Rust的所有权系统。**第5章**则介绍了结构体和方法。**第6章**涵盖了枚举、`match`表达式，以及`if let`和`let...else`这样的控制流结构。你将使用结构体和枚举来创建自定义类型。

在**第7章**中，你将了解Rust的模块系统以及用于组织代码及其公共应用程序接口（API）的隐私规则。**第8章**介绍了标准库提供的一些常见的集合数据结构：向量、字符串和哈希映射。**第9章**则探讨了Rust的错误处理理念和技术。

**第10章**深入探讨了泛型、特质和生命周期，这些概念让你能够定义适用于多种类型的代码。**第11章**则主要讨论测试，尽管Rust提供了安全保障，但测试仍然是确保程序逻辑正确的必要手段。**第12章**中，我们将自己实现`grep`命令行工具的一部分功能，该工具可以搜索文件中的文本。为此，我们将使用前几章中讨论过的许多概念。

**第13章**探讨了闭包和迭代器：这些都是源自函数式编程语言的Rust特性。在**第14章**中，我们将更深入地了解Cargo，并讨论与他人共享库文件的最佳实践。**第15章**则介绍了标准库中提供的智能指针以及实现其功能的特性。

在**第16章**中，我们将介绍不同的并发编程模型，并探讨Rust如何帮助您毫无畏惧地使用多个线程进行编程。在**第17章**中，我们将进一步深入研究Rust的异步和等待语法，以及任务、未来和流等概念，以及它们所支持的轻量级并发模型。

**第18章**探讨了Rust的惯用法与面向对象编程原则之间的比较。**第19章**则介绍了模式匹配的相关内容，这是在Rust程序中表达思想的有力手段。**第20章**涵盖了各种高级主题，包括不安全的Rust代码、宏的使用，以及关于生命周期、特性、类型、函数和闭包等方面的内容。

在**第21章**中，我们将完成一个项目，在这个项目中，我们将实现一个低级的多线程Web服务器！

最后，一些附录中包含了关于该语言的有用信息，这些内容以参考性更强的形式呈现。**附录A**介绍了Rust的关键字，**附录B**涵盖了Rust的运算符和符号，**附录C**介绍了标准库提供的可派生特性，**附录D**介绍了一些有用的开发工具，而**附录E**则解释了Rust的不同版本。**附录F**中提供了本书的翻译内容，而**附录G**则详细说明了Rust的构建过程以及当前版本的Rust是什么。

阅读这本书没有错误的方式：如果你想跳过某些章节，那就这么做吧！如果你遇到任何困惑，可能需要回到前面的章节去理解。但无论哪种方式都随你喜欢就好。
