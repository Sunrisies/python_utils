# Rust编程语言

由 Steve Klabnik、Carol Nichols 和 Chris Krycho 编写，并得到了 Rust 社区的贡献。

此版本的文本假设您正在使用 Rust 1.90.0（发布日期：2025-09-18）或更高版本。所有项目的 *Cargo.toml* 文件中应包含 ``edition = "2024"``，以便配置它们使用 Rust 2024 版本的惯用法。有关安装或更新 Rust 的说明，请参阅第1章的[“安装”部分][install]<!-- 忽略 -->，有关版本的更多信息，请参阅[附录E][appendix-e]<!-- 忽略 -->。

HTML格式可以在以下网址在线查看：  
[https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/)  
同时，通过安装了`rustup`的Rust环境，也可以离线访问该文档。运行`rustup doc --book`即可打开。

还有一些社区提供的[翻译版本]可供参考。

该内容以[平装本和电子书格式由No Starch Press提供]。[nsprust]
