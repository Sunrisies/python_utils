# 高级功能

到目前为止，你已经学习了Rust编程语言中最常用的部分。在继续第21章的另一个项目之前，我们将探讨一些你可能会偶尔遇到但可能不会每天使用的语言特性。你可以将这一章节作为遇到任何未知情况的参考。这里介绍的功能在非常特定的情况下非常有用。虽然你可能不会经常使用这些功能，但我们希望确保你能够掌握Rust所提供的所有功能。

在本章中，我们将涵盖以下内容：

- 不安全Rust：如何放弃Rust的一些保证，并手动负责维护这些保证
- 高级特性：关联类型、默认类型参数、完全限定语法、超特性以及与特性相关的新类型模式
- 高级类型：关于新类型模式的更多内容、类型别名、永不类型以及动态大小类型
- 高级函数和闭包：函数指针和返回闭包
- 宏：在编译时定义更多代码的几种方式

这是一系列适用于所有人的 Rust 功能！让我们开始吧！
