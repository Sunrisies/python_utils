# 常见的编程概念

本章介绍了几乎所有编程语言中都存在的概念，以及它们在Rust中的运作方式。许多编程语言在核心机制上有很多共同之处。本章所介绍的概念并非Rust所独有，但我们会在Rust的背景下对这些概念进行讨论，并解释使用它们的相关规范。

具体来说，你将学习变量、基本类型、函数、注释以及控制流。这些基础内容存在于每一个Rust程序中，及早掌握它们将为你提供一个坚实的基础。

> #### 关键字  
> Rust语言有一组被保留供语言本身使用的关键字，这与其他语言类似。需要注意的是，你不能将这些单词用作变量或函数的名称。大多数关键字都有特定的含义，你会在Rust程序中利用它们来完成各种任务；有一些关键字目前没有相关的功能，但它们被保留下来，以便将来可能在Rust中添加相关功能。你可以在[附录A][appendix_a]中找到所有关键字的列表。
